# Source Coverage

This is the live source map for the workbench. It describes what each configured source lane is for,
what it contributes today, and what still blocks a credible first civic-structure release. It is not
a changelog.

Use `deno task dc -- source list --json` for the machine-readable configured surface. Use
`deno task dc -- smoke structure`, `deno task dc -- smoke tier0`, and
`deno task dc -- smoke inventory` for clean temp-workbench rehearsals.

Tiers:

- Tier 0: needed for a credible first civic-structure release.
- Tier 1: high-value next improvement.
- Tier 2: useful completeness expansion.
- Tier 3: future or out of scope for now.

## Tier 0 Gaps

1. Keep DCGIS agency `BRANCH` and `MAYORAL_CLUSTER` taxonomy as evidence until source-backed
   structure modeling supports canonical fields or relationships.
2. Reconcile public-body overlaps across DCGIS BCC, Open DC, Quickbase, and ANC so duplicate bodies
   stay source-backed instead of being asserted silently.
3. Review Open DC legal-authority and governed-by candidates selectively; enabling-authority
   citations currently release as entity legal refs, not as non-exported legal relationships.
4. Normalize ambiguous legal refs only where the source evidence supports it, especially DCGIS
   reorganization-plan, act-name, and law-number citations.
5. Finish the remaining release-policy decisions around ANC tiering and Council role/seat shape
   before making a stronger first-release claim.

## Ambiguity-Heavy Surfaces

- `open_dc.public_bodies`: keep legal-authority and governing-agency rows review-first whenever the
  source is vague, the endpoint is unclear, or the bounded detail surface has not captured the
  needed body yet.
- `council.committees`: grouped or qualified oversight labels, funds, programs, regional bodies, and
  combined office/commission labels still need explicit source-backed entity decisions or a better
  normalization pass before acceptance.
- `mota.quickbase`: keep seat, status, authority, and appointee observation review narrower than the
  public-body comparison surface so appointment detail does not blur the structure lane.

Next lane recommendation: keep working the Tier 0 public-body reconciliation surface before widening
source coverage again.
